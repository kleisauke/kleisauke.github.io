Description

An app also includes a website. In this case, this website is designed to synchronize the timetables of the school timetable website.

The website uses the latest technologies to make the schedules more quickly appear on the app.

Users can view their schedule on rooster.rocrooster.net. For developers there is also an API available, this can be viewed at api.rocrooster.net.
