Beschrijving

Bij een app hoort natuurlijk ook een website. In dit geval is deze website opgezet om de roosters te synchroniseren van de lesrooster website.

De website maakt gebruik van de nieuwste technologieën om de lesroosters sneller te laten verschijnen op de app.

Gebruikers kunnen hun rooster bekijken op rocrooster.net/rooster. Voor ontwikkelaars is er tevens een API beschikbaar, deze kun je bekijken op api.rocrooster.net.
